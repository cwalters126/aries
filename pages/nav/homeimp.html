slowly transforming my millenial grey townhome into a cozy, colorful space.

Home Improvement 1

Cleaned up this Sears Typewriter I got off facebook marketplace. Replaced the ribbon (they sell purple ink!!).

This has been super fun for me to have around the house. I use it for letters, notes, stationery, etc.

A couple things I learned: this machine uses a lowercase 'l' as a 1. There is no 'plus' (+) sign, supposedly you can use a combination of dashes to create one, but not sure I have achieved that. The exclaimation point (!) uses similar logic; you use a period and the apostrophe.
